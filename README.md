

A full-stack AI-powered photo organizer that automatically labels uploaded images using Google Cloud Vision API and allows users to search photos by keyword.
Features
**
Upload images from a React frontend
AI-generated image labels using Google Cloud Vision API
Store image metadata in MongoDB
Search images using keywords
Responsive frontend with React + Vite
Express backend with modular architecture
Static image hosting for uploaded files
Graceful MongoDB and server shutdown handling

**

Tech Stack
Frontend
React
Vite
Axios
Backend
Node.js
Express
MongoDB
Mongoose
Multer
Google Cloud Vision API
